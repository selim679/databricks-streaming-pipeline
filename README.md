# Databricks Streaming Data Pipeline

## Project Overview

This project implements a real-time data pipeline using Databricks and Delta Lake.

##  Architecture

* Data ingestion (streaming)
* Bronze / Silver / Gold layers
* Data transformation with Spark
* Analytics

## 🛠️ Tech Stack

* Databricks
* Apache Spark
* Delta Lake
* Python

##  Use Case

Real-time data processing for analytics (IoT / logs simulation).

##  Status

Step 1: Project initialized
 Step 2: Streaming ingestion (Bronze layer)
Step 3: Data transformation (Silver layer completed)
Step 4: Aggregation (Gold layer)
